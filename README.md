## How to install
```bash
make build
make package-install
```

## How to use

#### Print help
```bash
gendiff -h
```
For chousing styles use -f or --format arg

#### Avalible styles:
* stylish output (```--format=stylish```)
* plain text output (```--format=plain```)
* json output (```--format=json```)
